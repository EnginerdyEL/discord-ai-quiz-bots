# Brainy — Daily Math Puzzle Bot

A Discord automation that posts a daily brain puzzle for engineering-minded math enthusiasts, along with the previous day's solution.

## What it does

Every day, Brainy automatically:
1. Posts the solution to yesterday's puzzle (answer behind a spoiler tag)
2. Generates a new puzzle using the Anthropic Claude API
3. Posts the new puzzle with a reminder to verify the answer yourself
4. Saves today's puzzle and solution to a GitHub Gist for tomorrow's reveal

## Puzzle categories

Puzzles are randomly selected from:
- Probability
- Geometry
- Logic
- Calculus
- Statistics
- Number Theory

## Tech stack

- **Python 3.14**
- **Anthropic Claude Sonnet** — puzzle generation with two-method verification
- **GitHub Gist** — persistent storage for daily puzzle state
- **requests** — webhook posting and Gist API calls
- **GitHub Actions** — daily scheduling (cron job, not yet configured)

## Setup

### Prerequisites
- Python 3.13+
- A Discord webhook URL for the target channel
- An Anthropic API key (console.anthropic.com)
- A GitHub personal access token with `gist` scope
- A GitHub Gist ID (create a secret gist with a `puzzle_state.json` file)

### Local development
```bash
git clone https://github.com/EnginerdyEL/math-puzzle-bot.git
cd math-puzzle-bot
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

Create a `.env` file in the root directory (never commit this):
```
ANTHROPIC_API_KEY=your_anthropic_api_key_here
GITHUB_TOKEN=your_github_personal_access_token_here
GIST_ID=your_gist_id_here
DISCORD_WEBHOOK_URL=your_webhook_url_here
```

Then run:
```bash
python3 puzzle.py
```

### Initial Gist setup

Create a secret Gist at gist.github.com with a file named `puzzle_state.json` containing:
```json
{
  "date": "",
  "category": "",
  "puzzle": "",
  "solution_steps": "",
  "solution_answer": ""
}
```

## Notes on answer accuracy

Puzzles are generated by Claude Sonnet using a two-method verification prompt. However, AI-generated solutions can occasionally be incorrect — always verify the answer yourself. The puzzle post includes a reminder to do so.

## Deployment

GitHub Actions cron job not yet configured — see Future Ideas below.

To deploy to a different Discord server, create a webhook in the target channel and update `DISCORD_WEBHOOK_URL` in your `.env` or GitHub secrets.

## Ongoing Cost and Maintenance

The bot leverages the Anthropic API to generate the puzzles and solutions, and each call costs money but a fraction of a penny. When developing, the developer added $5 to the account, which should last years even using the deeper-thinking Claude Sonnet model to ensure better solution accuracy
